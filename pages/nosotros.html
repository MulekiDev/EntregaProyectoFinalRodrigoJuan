¿Con qué insumos trabajamos?

Hay una gran variedad de filamentos para la impresión 3D FDM y cada uno tiene sus propias características y usos.
FDM (Fused Deposition Modeling, en inglés) es una tecnología de impresión 3D que utiliza un proceso de deposición de material fundido para crear objetos en tres dimensiones.
En el proceso de impresión FDM, se utiliza un filamento de plástico termoplástico, que se funde y se deposita en capas para construir el objeto. Estas impresoras utilizan una boquilla caliente para derretir el filamento, que se extruye y deposita capa por capa sobre la plataforma de impresión.

Los filamentos de uso más común son:
PLA (ácido poliláctico): es un filamento biodegradable y fácil de imprimir. Se utiliza principalmente para impresiones 3D de baja resistencia y proyectos de prototipado.

ABS (acrilonitrilo butadieno estireno): es un filamento resistente y duradero, pero también es más difícil de imprimir que el PLA. Se utiliza principalmente para impresiones 3D de mayor resistencia, como piezas de maquinaria y juguetes.

PETG (tereftalato de polietileno glicol): es un filamento resistente y duradero, similar al ABS, pero más fácil de imprimir. Se utiliza principalmente para impresiones 3D de alta resistencia, como piezas para el exterior de los coches.
